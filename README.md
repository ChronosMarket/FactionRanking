# FactionRanking
FactionRanking est un plugin fais par Veko pour ChronosMarket qui permet de faire un classement
faction détailler et presonnaliser.

Tout sera peronalisable en jeux et en config que ce sois pour les menu ou même les messages.

vous pourrez faire un classement de faction général, de farm, d'event et de kill avec ce système

Toujours en cours de développement.

Vesion de jeux : 1.8
